PRODIGY_WD_05 — Weather App
A weather web application built during my Web Development Internship at Prodigy InfoTech.

📌 Task Description
Build a web page that fetches weather data from a weather API based on the user's location or a user-inputted location. Display the current weather conditions, temperature, and other relevant information.

✨ Features

🔍 Search weather by city name
📍 Current location weather (GPS)
🌡️ Temperature & feels like
💧 Humidity, 💨 Wind speed
👁️ Visibility, 🌡️ Pressure
🌅 Sunrise & 🌇 Sunset times
🌤️ Weather condition with emoji
⚠️ Error handling for invalid cities
📱 Fully responsive dark UI


🛠️ Tech Stack

HTML5 — Structure
CSS3 — Styling & responsive design
JavaScript (Vanilla) — API calls, DOM updates
OpenWeatherMap API — Real-time weather data


💡 Key Concepts Learned

Fetch API for HTTP requests
Async/Await for asynchronous code
JSON data parsing
Geolocation API
Error handling with try/catch
Unix timestamp conversion
